## Usage Notes

- **Hero Image**: Use for blog header/hero section, should be 1920x1080px or larger
- **Feature Image**: Use within blog posts to illustrate the dashboard feature, should be 1200x675px or similar
- **Icon**: Use in navigation or as a favicon, should be 512x512px for web use, with SVG version for scalability

All images should maintain the "Frosted Glass" aesthetic with soft, translucent effects and modern, clean design language consistent with the extension's visual identity.
